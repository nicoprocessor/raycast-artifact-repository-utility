# Contributing

Thanks for your interest in contributing to Raycast Artifact Repository Utility.

## Development Setup

1. Install dependencies:
   - `npm install`
2. Start extension development:
   - `npm run dev`

## Quality Checks

Before opening a pull request, run:

- `npm run lint`
- `npm run build`
- `npm run test -- --run`

## Pull Request Guidelines

- Keep changes focused and minimal.
- Reuse existing patterns and provider abstractions.
- Update README when behavior changes.
- Ensure CI is green before requesting review.
